A modern, responsive website for a dating agency "My Match". Built with HTML5, Sass, and Bootstrap 5, it showcases the agency's services, team, and approach in a clean, user-friendly design.

The Russian version is available [here](README.ru.md).

## Description
This multi-page website presents a dating agency's brand, values, and offerings. It includes a home page with a carousel and statistics, an "About Us" page with team profiles and methodology, a "Services" page with pricing cards, and an application form. The project demonstrates semantic HTML, custom Sass styling, and Bootstrap components.

## Technologies Used
- **HTML5** – semantic markup
- **CSS3 / Sass** – modular styling with variables, mixins, and nesting
- **Bootstrap 5** – responsive grid, navbar, carousel, cards, forms
- **Font Awesome 6** – icons

## Features
- Fully responsive layout (mobile, tablet, desktop)
- Home page with image carousel, welcome section, and statistics sidebar
- "About Us" page with company values, team cards, and unique methodology
- "Services" page with three pricing plans and interactive hover effects
- "Submit Application" page with a validated HTML5 form
- Consistent header with navigation and footer across all pages
- Custom color scheme (#b1103a, #ffa4a4, #fffaed)
- Smooth animations and transitions on interactive elements

## Installation & Usage

### Requirements
- Any modern web browser.
- Optional: Sass compiler if you want to modify styles.

### Steps
1. Clone the repository:

  `git clone https://github.com/yourusername/my-match.git`
  
  Open any .html file in your browser (e.g., index.html).

2. To customize Sass styles:

  `npm install -g sass`
  
  `sass --watch scss/main.scss css/main.css`
